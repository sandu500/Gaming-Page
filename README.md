# Gaming Hub Landing Page

A premium gaming hub landing page built with clean HTML, CSS, and Vanilla JavaScript.

## Features
- Modern, dark-themed UI with glassmorphism.
- Responsive design for all devices.
- Smooth scroll animations and interactive navbar.
- Automated CI/CD pipeline using GitHub Actions.

## Structure
- `.github/workflows/`: GitHub Actions workflows (CI and Deployment).
- `src/`: Main source code.
    - `index.html`: Entry point.
    - `styles/style.css`: All styling.
    - `scripts/script.js`: Interactive logic.

## Hosting
### Vercel (Recommended)
This project is optimized for Vercel. Simply connect your GitHub repository to Vercel, and it will automatically deploy using the provided `vercel.json` configuration.

### GitHub Pages
The project also includes a GitHub Action in `.github/workflows/deploy.yml` for manual or automated deployment to GitHub Pages.
